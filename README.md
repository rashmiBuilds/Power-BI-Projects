# Power BI Projects

This repository contains Power BI dashboards created to analyze business and sales data using interactive visualizations.

These projects demonstrate data analysis, data visualization, and business intelligence skills using Power BI.

## Dashboards Included

### 1. Madhav Ecommerce Sales Dashboard
This dashboard analyzes the sales performance of an e-commerce business.

Key Insights:
- Total Sales Amount: 118K
- Total Profit: 12K
- Total Quantity Sold: 1409
- Average Order Value: 33K

Visualizations Used:
- Sales by State
- Quantity by Category
- Profit by Sub-Category
- Monthly Profit Analysis
- Customer-wise Sales
- Payment Mode Distribution

Purpose:
To understand customer purchasing behavior, profitable product categories, and regional sales performance.

### 2. Global Sales Dashboard
This dashboard provides insights into global sales performance across different countries and products.

Key Insights:
- Total Profit: 16.9M
- Total Sales: 118.7M
- Total Discounts: 9.2M
- Units Sold: 1.1M

Visualizations Used:
- Sales by Country
- Profit by Country
- Sales by Product
- Units Sold by Month
- Sales by Customer Segment
- Product Profit Comparison

Purpose:
To analyze international sales trends and product performance across different markets.
